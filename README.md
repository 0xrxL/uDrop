```uDrop``` was born as private C# project, under the name of ```uPatcher``` in mid-2022, after the ```Vanced Team``` closure.

Its main goal was (and still is) to create a modded YouTube app, based on my personal vision with a few, yet essential, mods (some of which were also developed by me), in order to make the app decently usable. In practice: the stock version that never was. ğŸ˜

```uDrop``` follows the old philosophy of the ```Vanced Team``` to build their mods: with a dedicated C# project.

In the end, ```uDrop``` doesn't want to compete with other Android patchers (it wouldn't be possible in any case, due to its different operating mechanisms).  
```uDrop``` simply aims to be a tribute to the past work of the ```Vanced Team``` (and, above all, a personal patcher I can use to test new YouTube beta versions with minimal effort).
